## PetLife - Instruções de Uso

1. **Clone o Repositório:**
   ```bash
   git clone https://github.com/seu-usuario/petlife-clinica-veterinaria.git
   ```

2. **Abra o Site:**
   ```bash
   cd petlife-clinica-veterinaria
   ```
   Abra o arquivo `index.html` no seu navegador preferido.

3. **Explore o Site:**
   Navegue pelo site.
